HTML

Hypertext Markup Language (HTML) is the standard markup Language forcreating webpages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the doccument.

HTML elements are the building blocks of HTML pages. WIth HTML constructs, images and other objects, such as interctive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, Lists , Links quotes and other items. HTML elements are delineated by tags, written using angel brackets.
